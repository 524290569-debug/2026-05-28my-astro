# 网站启动说明

## 本地启动

在项目根目录打开 VS Code 终端，然后运行：

```powershell
npm run dev
```

如果 PowerShell 提示 `npm.ps1` 脚本被禁止运行，可以改用：

```powershell
npm.cmd run dev
```

启动后在浏览器打开终端里显示的本地地址。通常是：

```text
http://127.0.0.1:4321/2026-05-28my-astro/
```

## 构建网站

生成静态部署文件：

```powershell
npm run build
```

如果 PowerShell 拦截 `npm`，使用：

```powershell
npm.cmd run build
```

## 本地预览构建结果

```powershell
npm run preview
```

或：

```powershell
npm.cmd run preview
```
