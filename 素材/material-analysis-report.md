## 所需素材层次结构

| 层次 | 主要内容 | 材料举例 |
| --- | --- | --- |
| **1. 天空/氛围层** | 昏暗天空、云层、雾气、雨丝等，提供整体氛围。 | 低饱和度的夜空贴图、乌云和雾气过渡素材。 |
| **2. 远景摩天楼** | 背景的城市天际线，轮廓柔和，灯光稀疏。 | 轮廓模糊的摩天楼群、遥远的霓虹灯。 |
| **3. 中景建筑群** | 中层建筑和城市层次，包含立交桥、广告牌、立管等。 | 有霓虹灯招牌的中层楼群、立体交错的桥梁和路轨。 |
| **4. 近景建筑/屋顶层** | 前景建筑屋顶、栏杆、平台、管道、电线等细节。 | 带天台护栏的楼顶、窗户、空调机、各种线路。 |
| **5. 街道/地面层** | 近景街道、潮湿的路面、积水和反光，增加纵深感。 | 窄巷道、雨后的街道、路灯、水坑反光等。 |
| **6. 前景装饰元素** | 电线、霓虹招牌、空调外机、栏杆等装饰物。 | 独立的霓虹招牌、警示牌、小型建筑设备。 |
| **7. 灯光/标识层** | 各种霓虹灯、广告屏幕、指示牌，用于点缀画面。 | 日文/英文招牌、酒吧标识、酒店灯牌、外卖广告屏。 |
| **8. 人物/物件层（可选）** | 可以加入小型人物或家具增强故事性，也可以为空，突出孤独氛围。 | 背影人物、废弃座椅、猫、小型植被等。 |

这些层次组合在一起，可以通过视差滚动或简单叠加形成具有深度感的多层网站头图。最底层的天空和远景负责构建基调，中景和近景提供主要的细节，而灯光、标识和装饰元素用来强调未来科技感和生活气息。
